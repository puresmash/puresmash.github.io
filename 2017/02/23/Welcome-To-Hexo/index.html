Welcome To Hexo

2017-02-23
Blog, Hexo

介紹

從 Java 轉來 JS 生態系有一段時間了，有部分疊床架屋的 JavaEE 技能開始印象薄弱也是很合理的一件事，但前陣子發現，一兩個月沒有修改的 JS Side Projects 熟悉度也開始直線下降，這一點我實在不能接受…。因此擱置很久的部落格撰寫計畫不能再拖，花了一段時間搞定了這個站。

之所以選用 Hexo 主要是因為，它是一個使用 NodeJS 引擎的 Static Render 工具，我可以

  • 使用 Markdown 編輯，書寫格式是通用的且不必綁定在任何平台
  • 由於是 JS Based 的，我可以方便的修改其 Template 兼練功
  • 方便的架設在 Github-Pages 上，不需額外尋找伺服器，撰寫文字時也可滿足填滿綠色框框的強迫症
